# Meshi Navi（メシナビ）

レシピ共有Webアプリケーション｜就活ポートフォリオ

## サービス概要

Meshi Naviは、ユーザーがレシピを投稿・閲覧・管理できるレシピ共有サービスです。
料理好きが集まり、オリジナルレシピをシェアできるプラットフォームを目指しました。

🌐 **本番URL**: https://yamada-portfolio.com

## 技術スタック

| 項目 | 技術 |
|------|------|
| バックエンド | PHP 8.3 / Laravel 12 |
| フロントエンド | Blade / CSS |
| データベース | SQLite（ローカル）/ MariaDB（本番） |
| 認証 | Laravel Breeze |
| 画像保存 | ローカルストレージ |
| インフラ | XServer |
| バージョン管理 | GitHub |

## 主な機能

- ユーザー登録・ログイン・ログアウト
- レシピ投稿・編集・削除（CRUD）
- 画像アップロード
- レシピ一覧・詳細表示
- 投稿者のみ編集・削除可能（認可）

## セキュリティ対策

- CSRF対策
- XSS対策（Bladeエスケープ）
- SQLインジェクション対策（Eloquent ORM）
- パスワードのハッシュ化（bcrypt）
- 認証・認可（Laravel Breeze）
- バリデーション
- ファイル形式チェック
- HTTPS通信
